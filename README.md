# 🎓 公益助学

> 扫码 → 看孩子 → 认领 → 实时追踪近况  
> **零服务器、零数据库费用、GitHub Pages 一键部署**

---

## 🚀 5 分钟部署指南

### 第一步：飞书应用（3 分钟）

1. 打开 [飞书开放平台](https://open.feishu.cn/) → 创建企业自建应用 → 名称填"公益助学"
2. 左侧「权限管理」→ 搜索 `bitable:app` → 开通
3. 左侧「版本管理与发布」→ 创建版本 → 发布
4. 「凭证与基础信息」→ 记下 **App ID** 和 **App Secret**

### 第二步：飞书多维表格（2 分钟）

1. 飞书中新建一个多维表格（名称随意）
2. 复制表格 URL 中的 BASE ID：
   ```
   https://xxx.feishu.cn/base/BCEbxxxxx?table=...
                            ↑ 这就是 APP_TOKEN
   ```
3. 创建 4 个子表（见下方），复制每个子表的 ID：
   - 进入子表 → 右上角 `...` → 复制表格 ID

### 第三步：填入配置（1 分钟）

编辑 `js/config.js`，把获取到的信息填进去：

```javascript
const FEISHU_CONFIG = {
  APP_ID: 'cli_axxxxxx',          // 飞书 App ID
  APP_SECRET: 'xxxxxxxxxxxxx',     // 飞书 App Secret
  BITABLE_APP_TOKEN: 'BCEbxxxxx',  // 多维表格 BASE ID
  TABLE_CHILDREN: 'tblxxxxx',      // 孩子信息表 ID
  TABLE_ADOPTIONS: 'tblxxxxx',     // 认领记录表 ID
  TABLE_UPDATES: 'tblxxxxx',       // 近况照片表 ID
  TABLE_THANKS: 'tblxxxxx',        // 感谢信表 ID
};

const ADMIN_PASSWORD = 'admin888'; // 管理后台密码，记得改
```

### 第四步：部署到 GitHub Pages（1 分钟）

1. 在 GitHub 新建仓库 → 上传整个项目文件夹
2. Settings → Pages → Source 选 `main` 分支 → Save
3. 等待几十秒，出现 `https://你的用户名.github.io/仓库名` 就是你的网址

### 第五步：生成二维码

用 [草料二维码](https://cli.im) 把这个网址生成二维码，打印贴出来。

## 🔧 日常使用

### 管理员（在网页 admin.html 操作）

1. 打开 `你的网址/admin.html`
2. 输入管理密码（默认 `admin888`，在 `config.js` 里修改）
3. **添加孩子**：切换到"添加孩子"标签，填写信息，上传照片，保存
4. **编辑孩子**：在"孩子管理"中点击编辑，修改后保存
5. **上传近况照片**：编辑某个孩子时，下方可直接上传
6. **添加感谢信**：同上，在下方输入文字或上传手写信照片

### 或者直接在飞书表格里操作

飞书多维表格本身就是完整的后台，你也可以直接在表格里增删改数据，**网页会实时显示最新内容**。

## ⚠️ 安全提示

- `APP_SECRET` 在前端代码中可见，请确保飞书应用**只开通了 `bitable:app` 这一个权限**
- 管理密码建议修改为强密码（在 `config.js` 中改 `ADMIN_PASSWORD`）
- 如需更高安全性，可改用 Vercel 部署（有 serverless 函数可隐藏密钥）
